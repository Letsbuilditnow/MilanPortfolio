PROJECTS

  • Milan's Netflix

    For this project, I created a web application that replicates the core features of the popular streaming service, Netflix.One of the main features of the application is that it only shows trailers for a given list of movies, rather than allowing users to watch the full movie. This was a necessary design decision that I made due to not having the rights to show the full movies on the platform.

  • The Universe

    For this project, I created a web application that showcases images of the universe and provides a description of each image. To do this, I used the NASA API, which allows users to access a wide variety of images and data from NASA's various missions and observatories. In the end, I was able to create a unique and informative web application that allows users to explore the beauty and complexity of the universe.

  • TICK IT

    For this project, I created a web application that allows users to create and manage their own todo lists, using Firebase for cloud functionality and user authentication. I built the app using React, and integrated it with Firebase to use Firebase's authentication service to allow users to create accounts and sign in to the app. The use of Firebase has added an additional layer of convenience and security for users.
